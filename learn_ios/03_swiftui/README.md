# SwiftUI 基础

本目录包含 SwiftUI 框架的学习资料和示例代码。

## 📚 学习内容

### 基础组件
- Text, Image, Button
- TextField, SecureField, Toggle
- Picker, Stepper, Slider

### 布局容器
- VStack, HStack, ZStack
- List, ForEach
- ScrollView, LazyVStack

### 导航
- NavigationStack
- NavigationLink
- Sheet, Alert

### 状态管理
- @State
- @Binding
- @StateObject
- @ObservedObject
- @EnvironmentObject

## 🎯 学习建议

1. **使用 Xcode Preview**: 实时查看 UI 效果
2. **从简单组件开始**: 逐步组合
3. **理解声明式编程**: 与 UIKit 的区别
4. **实践状态管理**: 理解数据流向
