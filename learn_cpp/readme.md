# Java开发者的C++学习之旅

这个仓库旨在帮助有Java经验的开发者系统性地学习C++，通过结构化的学习路径和实践项目逐步掌握C++核心概念和高级特性。

## 项目结构

```
learn_cpp/
├── 01_basics/                     # C++基础知识
├── 02_memory_management/          # 内存管理与指针
├── 03_oop/                        # 面向对象编程
├── 04_advanced_features/          # 高级特性
├── 05_stl/                        # 标准模板库
├── 06_modern_cpp/                 # 现代C++(C++11及以后)
├── 07_concurrency/                # 并发编程
├── 08_performance/                # 性能优化
├── 09_design_patterns/            # 设计模式实现
├── 10_project_templates/          # 项目模板
├── java_to_cpp/                   # Java到C++的知识迁移指南
└── resources/                     # 学习资源索引
```

## 使用指南

1. 按照学习路线图顺序学习各模块
2. 每个模块包含：
   - README.md：概念解释、最佳实践和常见陷阱
   - 示例代码：从基础到进阶的实现示例
   - 练习题：巩固所学知识的编程练习

3. 编译和运行示例：
   ```bash
   cd <module_directory>
   g++ -std=c++17 <example_name>.cpp -o <example_name>
   ./<example_name>
   ```
