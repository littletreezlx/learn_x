# Next.js 基础知识

本模块涵盖Next.js开发的基础概念和技能，是后续进阶学习的基础。

## 学习目标

1. 理解Next.js的基本原理和架构
2. 掌握页面创建和路由系统基础
3. 熟悉各种预渲染方式及其应用场景
4. 学会基本的数据获取方法
5. 使用静态资源和样式

## 内容目录

1. [项目创建与结构](./01-project-setup/README.md)
   - 使用create-next-app创建项目
   - 理解项目目录结构
   - 开发环境配置

2. [页面与路由基础](./02-pages-routing/README.md)
   - 基于文件系统的路由
   - 静态路由与动态路由
   - 路由导航与链接

3. [React基础回顾](./03-react-basics/README.md)
   - 组件与JSX
   - Props与State
   - 常用Hooks
   
4. [预渲染与数据获取](./04-data-fetching/README.md)
   - 服务端渲染(SSR)
   - 静态站点生成(SSG)
   - 增量静态再生成(ISR)
   - 客户端数据获取

5. [样式与资源管理](./05-styling/README.md)
   - CSS Modules
   - Tailwind CSS
   - 图片与字体资源
   - 布局组件

## 练习项目

- 个人简历网站：使用Next.js创建一个多页面简历网站，展示个人信息、技能和项目经验
- 博客列表页：实现一个博客文章列表页面，使用SSG预渲染文章内容
